icqb
===

运维管理工具

前言：
  分享，两年前做游戏运维写的云主机管理工具,用不到功能都去掉了,用过滤器Filter控制用户权限,根据用户权限动态生成菜单
  结合一些开源的网络控制器可做管理工具～希望对初学者有用！
  
框架
  * Django
    - Django 1.6 https://www.djangoproject.com/
  * Mysqldb
    - MySQLdb 1.2.3 http://www.codegood.com/archives/129
    
安装
  * SQL
    - temp/icqb_db.sql
  * Django 
    - pip install Django==1.6
  * Mysqldb
    - pip install MySQLd
